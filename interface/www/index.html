Well, what is git then?

Git is a powerful tool for version control and cooperative coding. But to many it seems very complicated and hard to wrap your head around. This leads people to avoid using it properly and in turn gives them even more trouble.

This site was part of a workshop on git at Chalmers where we tried to introduce people to git in a not so frightening way. The presentation can be found here.
